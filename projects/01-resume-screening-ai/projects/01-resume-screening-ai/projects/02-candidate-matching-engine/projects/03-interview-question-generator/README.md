# 🎤 Project 03 — Interview Question Generator

> Generate role-specific, structured interview questions using LLMs

---

## 🎯 Problem Statement

Inconsistent interviews lead to inconsistent hiring decisions. Interviewers often ask ad-hoc questions that don't cover all competencies. This tool generates a structured question bank tailored to any role.

## 💡 Solution

Input a Job Description and get a structured set of questions across:
- Technical skills
- Behavioural / situational (STAR method)
- Culture fit
- Role-specific scenarios

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| `LangChain` | LLM chaining |
| `OpenAI GPT-4` | Question generation |
| `Streamlit` | Web interface |
